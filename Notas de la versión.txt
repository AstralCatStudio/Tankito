Notas de la versión:
- Se recomienda jugar en navegador de PC. En móvil se puede jugar pero no iniciar el flujo de partida.
- Se pueden conectar dos jugadores de forma remota. Para obtener el código de partida se debe inspeccionar la página y acceder a la consola. El código de partida aparecerá en la consola una vez se elija el servidor.
- Se recomienda seleccionar el servidor de relay situado en Londres (Europa) para una mejor conectividad.
